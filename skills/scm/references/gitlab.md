# GitLab Playbook

## Identification Hints
- GitLab instances expose REST APIs and project metadata.

## Safe Checks
1. Inventory groups and projects with read-only queries.
2. Capture protected branches and merge request requirements.
3. Validate token scopes for least-privilege.

## Evidence Capture
- Project visibility, default branch, protection rules.
- Access token scope summary.
