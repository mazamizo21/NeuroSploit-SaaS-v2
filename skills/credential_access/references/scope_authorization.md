# Scope and Authorization

## Goals
1. Ensure credential access aligns with written authorization.
2. Minimize collection and avoid disruption.
3. Record authorization scope and expiry.

## Scope Rules
1. Use provided credentials and access only.
2. External targets: credential extraction requires explicit authorization.
3. Prefer offline analysis of authorized evidence.
4. Do not store raw secrets outside approved evidence storage.

## Evidence Checklist
1. Scope reference ID.
2. Authorization confirmation note.
3. Authorization expiry and approver identity.
