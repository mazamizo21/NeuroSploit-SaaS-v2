# Minimal Data Extraction

## Goals
- Prove impact with minimal data exposure.
- Prefer metadata over sensitive data.

## Safe Checks
- Extract database name and table list only.
- Sample a single row from a non-sensitive table.

## Indicators to Record
- Table and column names only when possible
- Masked values for any sample data

## Evidence Checklist
- Minimal extract evidence
- Redaction notes
