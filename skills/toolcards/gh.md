# gh Toolcard

## Overview
- Summary: GitHub CLI provides command-line access to GitHub APIs and workflows.

## Advanced Techniques
- Use `gh api` for custom queries and pagination.
- Validate token scopes before inventory.

## Safe Defaults
- Read-only queries by default; avoid repository changes unless authorized.
- Scope to authorized orgs and repos.
