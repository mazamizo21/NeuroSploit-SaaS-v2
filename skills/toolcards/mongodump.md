# mongodump Toolcard

## Overview
- Summary: mongodump is the MongoDB database dump tool for exporting data.

## Advanced Techniques
- Use `--archive` and `--gzip` for efficient captures.
- Scope to specific databases or collections.

## Safe Defaults
- Explicit authorization required for data export.
- Store exports in secure, access-controlled locations.

## Evidence Outputs
- outputs: evidence.json, findings.json (as applicable)
