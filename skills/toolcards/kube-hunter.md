# kube-hunter Toolcard

## Overview
- Summary: kube-hunter is a Kubernetes security scanner that tests clusters for known vulnerabilities.

## Advanced Techniques
- Use remote mode for external scanning.
- Output JSON reports for evidence capture.

## Safe Defaults
- Explicit authorization required for active scanning.
- Limit scan scope to in-scope clusters.

## Evidence Outputs
- outputs: evidence.json, findings.json (as applicable)
