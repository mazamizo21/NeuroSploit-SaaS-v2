# glab Toolcard

## Overview
- Summary: Glab is the GitLab CLI for interacting with GitLab APIs and projects.

## Advanced Techniques
- Use `glab api` for custom queries and pagination.
- Validate token scopes before inventory.

## Safe Defaults
- Read-only queries by default; avoid repository changes unless authorized.
- Scope to authorized groups and projects.

## Evidence Outputs
- outputs: evidence.json, findings.json (as applicable)
