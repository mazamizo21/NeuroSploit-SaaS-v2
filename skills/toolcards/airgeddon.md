# airgeddon Toolcard

## Overview
- Summary: airgeddon is a multi-use bash script for auditing wireless networks.

## Advanced Techniques
- Keep workflows modular and use only the minimal steps needed.
- Prefer passive checks before any active validation.

## Safe Defaults
- Authorized networks only; do not test production networks without written approval.
- Avoid active disruption unless explicitly authorized.

## Evidence Outputs
- outputs: evidence.json, findings.json (as applicable)
