# azure-devops-cli Toolcard

## Overview
- Summary: Azure DevOps CLI is an extension to Azure CLI for interacting with Azure DevOps services.

## Advanced Techniques
- Configure default organization and project for consistent inventory.
- Use read-only queries for repos, pipelines, and policies.

## Safe Defaults
- Read-only queries by default; avoid repository changes unless authorized.
- Scope to authorized organizations and projects.
