# hcxtools Toolcard

## Overview
- Summary: hcxtools provides utilities for capturing and converting WPA handshakes and PMKIDs.

## Advanced Techniques
- Use PMKID capture only on authorized networks.
- Convert captures into standardized formats for offline analysis.

## Safe Defaults
- Explicit authorization required for handshake or PMKID collection.
- Avoid deauth unless explicitly authorized.

## Evidence Outputs
- outputs: evidence.json, findings.json (as applicable)
