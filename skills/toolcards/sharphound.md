# sharphound Toolcard

## Overview
- Summary: SharpHound is the official BloodHound data collector that gathers Active Directory information for graph analysis.

## Advanced Techniques
- Use narrow collection methods to reduce noise and data volume.
- Align collection scope with approved domains and organizational units.

## Safe Defaults
- Avoid high-frequency collections on production networks.
- Collect only the minimum attributes needed for the approved analysis.

## Evidence Outputs
- outputs: bloodhound collection JSON/ZIP
