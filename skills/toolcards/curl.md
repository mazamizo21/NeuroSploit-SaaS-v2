# curl Toolcard

## Overview
- Summary: curl is a command-line tool for transferring data with URLs and supports HTTP APIs.

## Advanced Techniques
- Use headers and authentication options to validate access controls.
- Capture JSON responses for evidence.

- Scope rules: explicit target only.
