# tcpdump Toolcard

## Overview
- Summary: tcpdump captures network packets for troubleshooting and evidence collection.

## Advanced Techniques
- Use capture filters to limit scope and reduce noise.
- Rotate output files for long-running captures.

## Safe Defaults
- Capture only in-scope traffic and minimize sensitive data collection.
- Use short capture windows for production systems.

## Evidence Outputs
- outputs: evidence.json, findings.json (as applicable)
