# tor Toolcard

## Overview
- Summary: Tor is an anonymizing network overlay used to route traffic through relays.

## Advanced Techniques
- Combine with `proxychains` for controlled routing.
- Validate exit-node impact on target access and latency.

## Safe Defaults
- Use Tor only with explicit authorization and documented scope.
- Avoid using Tor for high-volume scans or time-sensitive operations.
