# tsql Toolcard

## Overview
- Summary: tsql is the FreeTDS CLI client for connecting to Microsoft SQL Server.

## Advanced Techniques
- Use configuration files to define server aliases and encryption settings.
- Run read-only metadata queries to enumerate databases and server version.

## Safe Defaults
- Rate limits: avoid repeated auth attempts on external targets.
- Scope rules: explicit target only.

## Evidence Outputs
- outputs: db_access.json
