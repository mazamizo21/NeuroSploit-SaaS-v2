# git Toolcard

## Overview
- Summary: Git is the distributed version control system used to manage source code repositories.

## Advanced Techniques
- Use shallow clones for metadata-only inventory.
- Use `ls-remote` for minimal-impact checks.

## Safe Defaults
- Read-only operations only unless explicitly authorized.
- Scope to authorized repositories and tokens.

## Evidence Outputs
- outputs: evidence.json, findings.json (as applicable)
