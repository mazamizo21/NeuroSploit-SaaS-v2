# psql Toolcard

## Overview
- Summary: psql is the PostgreSQL interactive terminal used for running queries and meta-commands.

## Advanced Techniques
- Use meta-commands to list databases, schemas, and tables quickly.
- Use script mode for repeatable, read-only enumeration.

## Safe Defaults
- Rate limits: avoid repeated auth attempts on external targets.
- Scope rules: explicit target only.

## Evidence Outputs
- outputs: db_access.json
