# tshark Toolcard

## Overview
- Summary: Tshark is the command-line interface to Wireshark for packet capture and analysis.

## Advanced Techniques
- Use display filters to extract specific evidence fields.
- Export JSON for structured reporting.

## Safe Defaults
- Capture only in-scope traffic and minimize sensitive data collection.
- Use short capture windows for production systems.

## Evidence Outputs
- outputs: evidence.json, findings.json (as applicable)
