# subfinder Toolcard

## Overview
- Summary: subfinder performs passive subdomain discovery from multiple sources and supports structured output.

## Advanced Techniques
- Enable all sources for deeper coverage on authorized scopes.
- Combine with DNS resolution to validate live subdomains.

## Safe Defaults
- Rate limits: passive discovery is preferred on external targets.
- Scope rules: explicit target only.

## Evidence Outputs
- outputs: subdomains.json
