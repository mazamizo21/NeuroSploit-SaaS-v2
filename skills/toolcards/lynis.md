# lynis Toolcard

## Overview
- Summary: Lynis is a security auditing tool for Unix-based systems that performs system configuration checks and provides hardening guidance.

## Advanced Techniques
- Run audits with explicit profiles to align to compliance goals.
- Export reports to structured output for evidence capture.

## Safe Defaults
- Read-only checks only.
- Scope to authorized hosts and avoid heavy scans on production without a window.

## Evidence Outputs
- outputs: evidence.json, findings.json (as applicable)
