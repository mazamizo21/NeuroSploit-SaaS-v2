# aircrack-ng Toolcard

## Overview
- Summary: aircrack-ng is a complete suite of tools to assess Wi-Fi network security.

## Advanced Techniques
- Prefer passive capture and channel validation before any active testing.
- Separate capture, analysis, and reporting steps for reproducibility.

## Safe Defaults
- Authorized networks only; do not test production networks without written approval.
- Avoid disruptive actions (deauth/jamming) unless explicitly authorized.

## Evidence Outputs
- outputs: evidence.json, findings.json (as applicable)
