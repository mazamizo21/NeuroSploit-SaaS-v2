# RDP Session Controls (Explicit-Only)

## Scope Gate
Interactive sessions require explicit authorization, especially for external targets.

## Controls to Validate (When Authorized)
1. Restricted Admin Mode (if policy allows).
2. Clipboard, drive, and printer redirection settings.
3. Multi-factor requirements or smart-card enforcement.
4. Session idle timeout and disconnect behavior.

## Evidence Checklist
1. Approved session start time and operator.
2. Host identity and session settings observed.
3. No data exfiltration during validation.
