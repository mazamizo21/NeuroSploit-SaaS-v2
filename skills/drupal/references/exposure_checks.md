# Exposure Checks

## Goals
1. Identify exposed config files, backups, or directory listings.
2. Record response codes and exposure scope.

## Safe Checks
1. Check for common backup filenames and changelog exposure.
2. Inspect robots.txt for sensitive paths.
3. Record redirects and authentication barriers.

## Evidence Checklist
1. Exposed paths and status codes.
2. Notes on sensitive exposure.
3. Response headers or body evidence.
