# Joomla Core Versioning

## Goals
1. Identify Joomla core version and update status.
2. Record evidence source and exposure notes.

## Safe Checks
1. `joomscan -u https://target`
2. Check `administrator/manifests/files/joomla.xml` if exposed.

## Evidence Checklist
1. Core version and release notes (if available).
2. Update status.
3. Exposed manifest or metadata evidence.
